# Bank Marketing Prediction

## Overview

This project focuses on predicting whether a customer will subscribe to a term deposit based on historical marketing data. It uses machine learning techniques to analyze customer attributes and campaign-related features to improve targeting strategies.

The goal is to help businesses identify potential customers and optimize marketing efforts using data-driven insights.

---

## Features

* Data preprocessing and cleaning
* Handling missing values and class imbalance (SMOTE)
* Feature engineering for improved model performance
* Classification using Decision Tree
* Model evaluation using accuracy, precision, recall, and ROC-AUC

---

## Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib, Seaborn

---

## Model Performance

* Achieved approximately 76% accuracy
* Evaluated using precision, recall, and ROC-AUC for reliability

---

## Dataset

The dataset used is the Bank Marketing dataset, which contains customer demographic and campaign-related information.
